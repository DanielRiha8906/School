Jednovýběrové testy nejsou tak klasické, většinou třeba jenom opakování historických měření

Dvouvýběrové testy - více časté -> porovnávám dva nezávislé výběry 
zobecnění jednovýběrové varianty -> komplikuje semi to tím, že mám dvě možnosti:
Dvouvýběrový t-test pro shodné roptyly
Welchův dvouvýběrový test pro různé rozptyly

Testované hypotézy obou testů jsou : 
H0 - střední hodnota x - střední hodnota Y = 0
H0 - střední hodnota x - střední hotnota Y !=0, < or > než 0 


Jaký t-test mám vybrat, na to potŘebuji shodu rozptylů -> Buď udělám Boxplot, ale okometrická metoda není úplně gucci -> F-test
F = Var(x)/Var(y) - Fn1-1,n2-1
n1 je rozsah výběru x, n2 Y
H0: rozptyly se ve výběrech neliší
h1: rozptyly se liší
P <= alfa => Zamítám H0, platí H1
P > alfa => Nezámítám H1

Wilcoxonův dvouvýběrový test pro porovnání dvou nezávislých výběrů, které nesplňují normalitu
H0: stŘední hodnota X - střední hodnota Y =  0
H1: stŘední hodnota X - střední hodnota Y != 0, < or > 0

Test je založen an pořadích hodnot sdruženého výběru. Postup: 
oba výběry smíchám dohromady, uspořádám dle velikosti, všechny pošlu zpátky, průměr se spočítá z pozic. Když to vyjde hodně jinak -> výsledek


Porovnáváme-li více než dva nezávislé výběry, používá se analýza rozptylu. 
Testuji:
H0: všechny střední hodnoty jsou stejné
H1: alespoň jedna stŕední hodnota se liší
